Formzin

Plugin do Jquery para facilitar o tratamento de formulários.

Baixar o Plugin Versão 0.13

Baixar o Plugin Versão 0.13 (min)

Formatação

Formatação de campos colocando apenas o nome da classe no campo do formulário com diversos tipos de formatações e mascaras.

Validação

Validação de formulários em tempo real com diversos tipos de validações.

Manipulação

Manipulação de campos nos formulários sem a necessidade de códigos javascript.

Como Utilizar

Para utilizar basta chamar o arquivo .js do plugin:

<script src="jquery.min.js" type="text/javascript"></script>
<script src="formzin.min.js" type="text/javascript"></script>

Para chamar o plugin em lightbox, dialog, ajax e etc:

<script type="text/javascript">
    Formzin.iniciar();                    
</script>

Formatar CEP (class="buscar_endereco")

Os atributo (prefixo) para colocar o prefixo do campo, para ser usado com diversos endereço. Ex: endereço de entrega e cobrança em Loja Virtual 
<input type="text" prefixo="en_" class="buscar_endereco" />

BAIRRO <input type="text" class="en_bairro" />
CIDADE <input type="text" class="en_cidade" />
UF <input type="text" class="en_uf" />
