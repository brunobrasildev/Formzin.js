Formzin

Plugin do Jquery para facilitar o tratamento de formulários.

Baixar o Plugin Versão 1.0.3

Baixar o Plugin Versão 1.0.3 (min)

Formatação

Formatação de campos colocando apenas o nome da classe no campo do formulário com diversos tipos de formatações e mascaras.

Validação

Validação de formulários em tempo real com diversos tipos de validações.

Manipulação

Manipulação de campos nos formulários sem a necessidade de códigos javascript.

Como Utilizar

Para utilizar basta chamar o arquivo .js do plugin:

<script src="jquery.min.js" type="text/javascript"></script>
<script src="formzin.min.js" type="text/javascript"></script>

Para chamar o plugin em lightbox, dialog, ajax e etc:

<script type="text/javascript">
    Formzin.iniciar();                    
</script>

CAMPO SELECT (class="selecao", data-classe_selecao="sel", data-selecao="1|div-azul,2|div-vermelha")

<select class="form-control selecao" data-classe_selecao="sel" data-selecao="1|div-azul,2|div-vermelha" id="sel">
    <option value="">Selecione</option>
    <option value="1">Mostrar div azul</option>
    <option value="2">Mostrar div vermelha</option>
</select>

DIV PARA MOSTRAR

<div class="sel div-azul" style="background: blue; color: #fff; padding: 5px; display: none;">
  <h5>AZUL</h5>
</div>
                        
<div class="sel div-vermelha" style="background: red; color: #fff; padding: 5px; display: none;">
   <h5>VERMELHO</h5>
</div>

Atributos:

data-classe_selecao="sel" Colocar a classe para todas as divs da seleção.
data-selecao="1|div-azul,2|div-vermelha" Atributo seleção para colocar value|class_para_mostrar. Podendo colocar mais de uma opção separado por virgula.

CAMPO RADIO (class="selecao")

<input type="radio" name="tipo_pessoa" class="selecao" data-classe_selecao="tp-radio" data-selecao="J|radio-dados-empresa" value="F" /> FISICA
<input type="radio" name="tipo_pessoa" class="selecao" data-classe_selecao="tp-radio" data-selecao="J|radio-dados-empresa" value="J" /> JURIDICA

DIV PARA MOSTRAR

<div class="tp-radio radio-dados-empresa" style="display: none;">
   <h5>Dados da Empresa</h5>
   <div class="form-group">
      <label for="razao-social">Razão Social</label>
      <input type="text" class="form-control" id="razao-social" />
   </div>
   <div class="form-group">
      <label for="cnpj">CNPJ</label>
      <input type="text" class="form-control" id="cnpj" />
   </div>                                
</div>

Atributos:

data-classe_selecao="tp-radio" Colocar a classe para todas as divs da seleção.
data-selecao="J|radio-dados-empresa" Atributo seleção para colocar value|class_para_mostrar. Podendo colocar mais de uma opção separado por virgula.

CAMPO CHECKBOX (class="selecao")

<input type="checkbox" name="formulario" class="selecao" data-classe_selecao="form" data-selecao="1|formulario" value="1" />

DIV PARA MOSTRAR

<div class="form formulario" style="display: none;">
   <h5>Meu Formulário</h5>

   </div>                                
</div>

Atributos:

data-classe_selecao="form" Colocar a classe para todas as divs da seleção.
data-selecao="1|formulario" Atributo seleção para colocar value|class_para_mostrar. Podendo colocar mais de uma opção separado por virgula.
